# Princípios de UX

**Fontes canónicas:** [`docs/design-system/design-system-v2.md`](../../docs/design-system/design-system-v2.md), [`docs/ui-audit/ui-audit-v2.md`](../../docs/ui-audit/ui-audit-v2.md)

## Princípios

1. **Clareza em dados sensíveis:** mascarar onde necessário (Clarity, etc.).
2. **Tema coerente:** light/dark com contraste legível (ver auditorias UI).
3. **Estados vazios úteis:** sempre próximo passo claro.
4. **Erros acionáveis:** sem stack traces ao utilizador.

## Anti-padrões

- “Dark mode” com texto ilegível.
- Prometer output sem mostrar limitações.
