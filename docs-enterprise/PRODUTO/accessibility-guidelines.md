# Diretrizes de acessibilidade

**Fonte canónica visual:** [`docs/design-system/components.md`](../../docs/design-system/components.md)

## Checklist mínimo (WCAG-minded)

- [ ] Contraste texto/fundo em ambos os temas
- [ ] Foco visível em botões e links (`focus-ring` onde aplicável)
- [ ] Labels em inputs críticos
- [ ] Não depender só de cor para estado de erro

- VoiceOver spot-check em Mac quando possível.
