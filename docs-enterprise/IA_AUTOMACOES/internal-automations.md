# Automações internas (roadmap seguro)

> Automações **fora** do produto (ex.: Notion, Zapier, scripts internos) — não confundir com pipelines do app.

## Candidatos de baixo risco

- Resumo semanal de métricas Stripe → email interno
- Alertas de uptime (Vercel / status pages)

## Evitar sem revisão legal

- Automação que copia dados de utilizadores para ferramentas não DPA-covered.
