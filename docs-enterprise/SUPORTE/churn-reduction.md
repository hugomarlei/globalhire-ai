# Redução de churn (playbook)

## Sinais precoces

- Queda de logins após primeira semana
- Falhas repetidas no gerador
- Tickets de insatisfação com output

## Acções

1. Resposta proactiva com dicas de prompt (sem culpabilizar)
2. Oferta de crédito **só** se política financeira permitir
3. Bug fix prioritário se causa técnica
