# Macros de resposta (templates)

> Personalizar sempre com nome e contexto. Não copiar outputs de IA do utilizador para tickets públicos sem consentimento.

## Atraso na entrega

```
Olá [Nome],

Obrigado por contactar a GlobalHire AI. Estamos a analisar o teu caso.

Enquanto isso, podes confirmar:
1. Se o pagamento apareceu como concluído no teu banco?
2. Se estás com sessão iniciada na mesma conta?

Cumprimentos,
Equipa GlobalHire AI
```

## Problema de qualidade do output

```
Olá [Nome],

Lamentamos que o resultado não tenha correspondido às expectativas.

Para ajudarmos melhor, indica:
- Que tipo de documento geraste (CV, carta, etc.)
- O que esperavas vs o que obtiveste (sem partilhar dados sensíveis que não queiras)

Cumprimentos,
Equipa GlobalHire AI
```
