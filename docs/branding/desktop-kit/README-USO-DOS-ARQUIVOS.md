# GlobalHire AI Brand Kit — Uso dos Arquivos

Esta pasta reúne os arquivos prontos para atividades manuais de marca, LinkedIn, apresentação institucional e validação com parceiros.

## Arquivos principais

- `linkedin-banner.png`: banner padrão para a página da empresa no LinkedIn.
- `linkedin-banner-dark.png`: versão escura para LinkedIn e posts institucionais.
- `linkedin-banner-light.png`: versão clara para apresentações, documentos e parceiros.
- `linkedin-banner.svg`: versão editável do banner.
- `favicon.svg` e `og-image.svg`: assets atuais do produto, quando disponíveis.
- `COPY-PRONTA-PARA-USO.md`: textos curtos, descrições e pitches para copiar.
- `LINKEDIN-CONTEUDO-PRONTO.md`: About, serviços, posts e hashtags para LinkedIn.
- `RELATORIO-FINAL.md`: resumo do que foi criado e próximos passos.
- `social-kit/`: ideias e formatos para posts, stories e anúncios.

## Onde usar cada banner

- LinkedIn company page: use `linkedin-banner.png`.
- Post fixado ou anúncio institucional: use `linkedin-banner-dark.png`.
- Documento para advogado, investidor ou designer: use `linkedin-banner-light.png` ou o SVG editável.

## Texto para LinkedIn

Copie a seção “About section em português” de `LINKEDIN-CONTEUDO-PRONTO.md` para a página da empresa. Se quiser posicionamento internacional, use também a versão em inglês.

## Texto para o site

Use em ordem:

1. Headline: “Pare de enviar currículos ignorados.”
2. Subheadline: “Crie candidaturas internacionais otimizadas para ATS, com currículo, carta de apresentação e LinkedIn adaptados à vaga em minutos.”
3. Tagline: “Get Hired Smarter.”

## Primeiros posts recomendados

1. Por que currículos genéricos falham em vagas internacionais.
2. O que é ATS Score e como interpretar sem paranoia.
3. Como adaptar currículo para uma vaga sem inventar experiência.

## Como manter consistência visual

- Use fundo escuro, texto claro e Signal Green `#32E875`.
- Não use imagens genéricas de RH.
- Não prometa emprego garantido.
- Use uma ideia por post.
- Assine com “GlobalHire AI · Get Hired Smarter.”

## O que ainda precisa ser feito manualmente

- Subir o banner no LinkedIn.
- Atualizar About da empresa.
- Cadastrar serviços.
- Publicar os primeiros posts.
- Validar o banner em desktop e mobile.
- Decidir se haverá designer para lapidar logo final.

## Arquivos para advogado, investidor ou designer

- Para advogado: pode enviar descrições institucionais e relatórios legais, mas não precisa enviar posts.
- Para investidor: use `COPY-PRONTA-PARA-USO.md`, banner e brand guidelines.
- Para designer: envie SVG, PNGs, brand guidelines e design system.
