# GlobalHire AI — Color System V2

Direção: premium SaaS, off-black, emerald sofisticado e acentos frios discretos.  
Objetivo: remover sensação neon/cyber e criar base visual confiável para IA, carreira global e produto escalável.

## Uso proibido

- Não usar verde neon dominante.
- Não aplicar glow forte em grandes áreas.
- Não usar arco-íris, cyberpunk ou gradientes infantis.
- Não usar verde como fundo de páginas inteiras.

## Contraste

- Texto principal `#F4F7F5` sobre `#070A0D`: alto contraste.
- Texto secundário `#A7B0B8` sobre `#070A0D`: adequado para UI de apoio.
- CTA `#2FBF8F` com texto `#070A0D`: alto contraste e melhor sobriedade que neon.

## Exemplos

- Botão principal: fundo `primary`, texto `primary-foreground`.
- Card: fundo `card`, borda `border`, texto `card-foreground`.
- Insight/score: `primary`.
- Dados ou visual IA: `secondary`, sempre com baixa opacidade.
