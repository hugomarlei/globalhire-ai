# GlobalHire AI Brand Kit V2 — Uso dos Arquivos

## Estrutura

- `logos/`: símbolo, wordmarks, favicon e app icon.
- `linkedin/`: banners LinkedIn V2 em PNG e SVG.
- `social-kit/`: posts e calendário inicial.
- `copy/`: textos prontos para site, pitch e campanhas.
- `website/`: assets para site e OG image.
- `legal-investor/`: resumos para advogado e investidor.
- `exports/`: arquivos prontos para envio externo.

## Qual arquivo usar no LinkedIn

- Banner principal: `linkedin/linkedin-banner-v2.png`.
- Se quiser fundo claro: `linkedin/linkedin-banner-v2-light.png`.
- SVG editável: `linkedin/linkedin-banner-v2.svg`.

## Qual usar como logo

- Avatar/company icon: `logos/logo-symbol.png`.
- Apresentações horizontais: `logos/logo-horizontal.png`.
- Materiais internos ou capa: `logos/logo-vertical.png`.
- Favicon: `logos/favicon.svg`.

## Textos para copiar

- Página da empresa: `LINKEDIN-CONTEUDO-PRONTO.md`.
- Site e pitch: `copy/brand-copy-v2.md`.
- Primeiros posts: `social-kit/linkedin-posts-v2.md`.

## O que NÃO usar mais da V1

- Banners antigos sem `v2` no nome.
- Símbolo anterior baseado em sparkle/monograma.
- Verde neon dominante.
- Copy longa no banner.

## Arquivos para advogado

Enviar `legal-investor/BRAND-RESUMO-PARA-ADVOGADO.md` e, se necessário, os documentos legais de `docs/legal`.

## Arquivos para investidor

Enviar `legal-investor/BRAND-RESUMO-PARA-INVESTIDOR.md`, `logos/logo-horizontal.png`, `linkedin/linkedin-banner-v2.png` e copy curta.

## Arquivos para designer

Enviar todos os SVGs, brand guidelines, color system e design system.

## Arquivos internos

Relatórios, auditorias, UI audit e implementação visual são documentação interna de produto.
