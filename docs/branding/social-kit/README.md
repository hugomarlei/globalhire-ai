# GlobalHire AI — Social Media Kit

## Direção visual

- Fundo escuro Ink com grid sutil.
- Signal Green como destaque.
- Texto grande, uma ideia por peça.
- Marca discreta no rodapé.
- Evitar fotos genéricas de escritório ou RH.

## Templates de LinkedIn posts

### Template 1 — Insight

Título grande: “Seu currículo não precisa agradar todo mundo.”  
Subtexto: “Ele precisa conversar com a vaga certa, no idioma certo, com sinais que ATS e recrutadores entendem.”  
Assinatura: GlobalHire AI · Get Hired Smarter.

### Template 2 — Antes/depois

Antes: “Responsável por relatórios.”  
Depois: “Transformou dados comerciais em insights executivos para decisões de receita.”  
Nota: “A IA não deve inventar fatos. Ela deve revelar melhor o valor real.”

### Template 3 — Checklist

Título: “Antes de aplicar para uma vaga internacional”  
Itens: idioma, país-alvo, palavras-chave, escopo, impacto, LinkedIn.

## Instagram feed

- 1080x1080.
- Margem segura de 96px.
- Título com até 9 palavras.
- Rodapé com logo e tagline.

## Stories

- 1080x1920.
- Título no primeiro terço.
- CTA no último terço.
- Usar enquete: “Seu currículo passa por ATS?”

## Thumbnails

- Usar score visual, como “42% → 91%”.
- Não usar promessa de emprego.
