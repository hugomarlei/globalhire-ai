# GlobalHire AI — Typography V2

## Fonte principal

Stack recomendada:

`Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

Motivo: aparência SaaS moderna, excelente legibilidade, suporte amplo e integração simples com Next.js.

## Fonte mono opcional

`ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", monospace`

Uso: dados técnicos, tokens, códigos, pequenos labels de sistema.

## Escala tipográfica

| Uso | Desktop | Mobile | Peso | Line-height |
|---|---:|---:|---:|---:|
| Hero H1 | 64-72px | 44-48px | 650-750 | 1.02 |
| Section H2 | 32-40px | 28-32px | 650-750 | 1.12 |
| Card title | 18-24px | 18-22px | 600-700 | 1.2 |
| Body | 15-17px | 14-16px | 400-500 | 1.6 |
| UI label | 12-14px | 12-14px | 500-650 | 1.4 |
| Caption | 11-12px | 11-12px | 400-500 | 1.45 |

## Letter spacing

- Padrão: `0`.
- Uppercase labels: até `0.04em`.
- Não usar letter spacing negativo.

## Regras mobile

- Evitar linhas com mais de 9 palavras em títulos.
- Botões com texto curto.
- Altura mínima de toque: 44px.
- Não reduzir body abaixo de 14px.
