# SOP Support

Status: PARCIAL  
Owner atual: Solo founder / operador técnico  
Última revisão: 2026-05-10

## Objetivo

Definir procedimento mínimo para suporte ao usuário.

## Canais

IMPLEMENTADO/PARCIAL:

- Contato público: `contato@globalhireai.com.br`.
- Privacidade: `privacy@globalhireai.com.br`.

PENDENTE:

- Help desk formal.
- SLA público.
- Macros de atendimento.

## Classificação

| Tipo | Exemplo | Prioridade |
| --- | --- | --- |
| Acesso | login, reset senha | Alta |
| Pagamento | cobrança, plano, cancelamento | Alta |
| IA | resultado ruim, erro de geração | Média |
| Produto | dúvida de uso | Média |
| Sugestão | melhoria futura | Baixa |

## Procedimento

1. Confirmar e-mail da conta.
2. Entender problema sem solicitar dados sensíveis desnecessários.
3. Verificar logs técnicos sem copiar currículo completo.
4. Reproduzir, se possível.
5. Responder com solução clara.
6. Registrar bug se houver falha real.

## Regras

- Nunca pedir senha.
- Nunca pedir dados de cartão.
- Não prometer aprovação em vaga.
- Não copiar currículo completo para ferramentas externas.
