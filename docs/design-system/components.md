# GlobalHire AI — Component Guidelines

## Buttons

Primary buttons use Signal Green with Ink text. Secondary buttons use translucent backgrounds with white borders. Avoid decorative gradients inside buttons.

## Cards

Cards use glass surface, 8px radius, subtle border and light internal highlight. Cards should frame repeated items, tools, pricing and dashboard modules.

## Inputs

Inputs must be dark, full width and clearly focusable. Sensitive fields must keep `data-clarity-mask`.

## Navigation

Public navigation should always expose “Entrar” and “Criar conta”, including mobile. App navigation can use grouped dropdowns for tools, analysis, documents and account.

## Dashboard

Dashboard should prioritize:

- plan status;
- usage;
- latest activity;
- quick actions;
- stats.

Avoid marketing copy inside authenticated product screens.

## Landing

Landing should combine:

- strong headline;
- product proof/mock;
- honest limitations;
- pricing;
- FAQ;
- legal footer.

## Dark mode

Dark mode is the primary brand environment. Light assets exist for external channels but product UI should remain dark-first.
