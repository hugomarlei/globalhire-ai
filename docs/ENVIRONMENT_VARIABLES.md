## Stripe

O projeto usa os nomes atuais:

- `NEXT_PUBLIC_STRIPE_STARTER_PRICE_ID`
- `NEXT_PUBLIC_STRIPE_PRO_PRICE_ID`
- `NEXT_PUBLIC_STRIPE_ELITE_PRICE_ID`

Não trocar para `STRIPE_PRICE_STARTER` sem refatorar o código. Os valores devem começar com `price_`.

## Produção

Use:

```bash
NEXT_PUBLIC_APP_URL=https://globalhireai.com.br
```

Nunca coloque secrets em variáveis `NEXT_PUBLIC_`.
